# Preparation - Setup Bruno API client

To run the integration scenarios, we use the open source API client Bruno for which we have prepared a collection. The collection contains sample requests and environment parameters to authenticate at the provided tenant.

If you run the exercises on the laptops provided by us, the Bruno client has been already installed. Otherwise, if you run the exercises on your own laptop, you can download and install the Bruno API client from [here](https://docs.usebruno.com/get-started/bruno-basics/download).

Optionally, you may use the provided Postman collection.

## Download the collection

In the SAP Integration Suite landing page, navigate to **Design > Integrations and APIs**. Select the integration package **Getting Started Tutorial - Solution** and switch to tab **Documents**. From there you can download the Bruno collection.

<br>![](/exercises/ex0/images/00_Bruno_00.png)

## Upload the collection

1. Open Bruno and select **Import Collection** either from the menu or from the home screen.

<br>![](/exercises/ex0/images/00_Bruno_01.png)

2. On the upcoming dialog, select **choose file(s)**.

<br>![](/exercises/ex0/images/00_Bruno_02.png)

3. Navigate to the location where you saved the beforehand downloaded collection, select the file **Getting Started.zip**, and select **Open**.

<br>![](/exercises/ex0/images/00_Bruno_03.png)

4. Browse to define a location, eventually create a new folder. Then select **Import**.

<br>![](/exercises/ex0/images/00_Bruno_04.png)

5. The new collection should show up in the navgation pane of Bruno. If you expand it, it should have one request.

<br>![](/exercises/ex0/images/00_Bruno_05.png)
