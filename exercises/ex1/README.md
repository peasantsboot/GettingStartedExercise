# Create an integration package and an integration flow

## Create an integration package

You first need to create an integration package where your new integration flow is created.

1. Open the SAP Integration Suite landing page, navigate to **Design > Integrations and APIs**, and select  **Create**.

<br>![](/exercises/ex1/images/01_Package_01.png)

2. In the **Header** tab of your new integration pacakge, fill in the **Name** of the integration package, e.g. **User XX** where **XX** is your user number from 00 to 45. Furthermore, maintain a short **Description**. The technical name is automatically set. Then click **Save**.

<br>![](/exercises/ex1/images/01_Package_02.png)

## Create an integration flow

1. Switch to the **Artifacts** tab, and select **Add > Integration Flow** from the menu.

<br>![](/exercises/ex1/images/02_IntegrationFlow_01.png)

2. In the upcoming dialog, enter a **Name** for the integration flow and choose **Add and Open in Editor**.

<br>![](/exercises/ex1/images/02_IntegrationFlow_02.png)

3. In the integration flow editor, choose **Edit** to start editing the integration flow.

<br>![](/exercises/ex1/images/02_IntegrationFlow_03.png)

## Summary

Now that yuo have created a new integration pacakge and a new integration flow, you are all set to implement the integration logic.
